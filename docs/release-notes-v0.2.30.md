## 必应每日壁纸库 v0.2.30

本版本新增全局分辨率选择，并增强“已下载的壁纸”画廊的信息展示。

### 全局分辨率

- 左侧导航栏新增“全局分辨率”，可选择“原图 UHD / 4K-3840×2160 / 2K-2560×1440 / 1K-1920×1080”。
- 默认仍为 4K-3840×2160，保持旧版本下载质量不变。
- 下载壁纸、批量下载和一键设置桌面壁纸都会使用当前选择的全局分辨率。
- “原图 UHD”会使用 Bing 无缩放查询参数的原始 UHD 文件地址；4K/2K/1K 会使用对应尺寸参数。

### 下载中心

- “下载中心 · 已下载的壁纸”卡片现在会显示本地图片分辨率与文件大小，例如 `3840×2160 · 1.93 MiB`。
- 支持读取 JPG / PNG / WebP 文件头获取尺寸；解析失败时仍会显示文件大小。

### 发布与下载

- 继续使用 GitHub Release + Gitee Release 双发布源。
- 国内更新优先下载 Gitee Release 附件，失败后自动回退 GitHub Release 附件。
- 海外或 GitHub 可访问环境可直接使用 GitHub Release 同名附件。
