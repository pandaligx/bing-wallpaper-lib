## 必应每日壁纸库 v0.2.33

本版本修复 2024 年 1 月 26 日北鹰鸮壁纸地址损坏的问题，并新增多语言界面与中英文项目文档。

### 壁纸归档修复

- 单独修复 `2024-01-26` 北鹰鸮壁纸，正确 OHR 标识为 `OHR.HawkOwl_ZH-CN3401920167`。
- 预览、1080p 与 4K 地址均恢复为正确分辨率参数，4K 地址使用 `3840×2160`。
- 修复记录写入独立 corrections 表，后续每四小时自动同步上游归档时仍会持续覆盖错误记录。
- 新增回归测试，锁定日期、中文标题、OHR 标识和 4K 地址。
- 内置历史快照已合并 GitHub 最新自动同步数据，更新至 2026 年 7 月 17 日。

### 多语言界面

- 新增 i18n 语言模块与持久化语言设置。
- 支持跟随 Windows 系统语言，以及简体中文、English、日本語、한국어、Русский、Français。
- 左侧栏设置按钮右侧新增国旗语言按钮，可直接展开语言菜单。
- 切换语言后立即刷新主要界面与系统托盘菜单，并在下次启动时保持选择。
- 旧版本设置文件没有语言字段时自动使用“跟随系统”，无需重置配置。

### 项目文档

- `README.md` 改为默认英文主页。
- 新增 `README.zh-CN.md` 简体中文主页，两种语言可在页面顶部互相切换。
- 删除三个已经过时的早期版本发布说明文件。

### 验证

- `cargo check`
- `cargo test`：43 项全部通过
- `cargo clippy --all-targets -- -D warnings`
- `cargo build --release`

### 发布与下载

- GitHub Release 与 Gitee Release 均提供 `bing-wallpaper-lib-v0.2.33-x64.exe`。
- 国内更新优先使用 Gitee Release 附件，失败后自动回退 GitHub Release 附件。
