## 必应每日壁纸库 v0.2.31

本版本修复历史壁纸缺失和近期日期偏移，并强化本地缩略图、aria2 RPC 与应用内更新流程。

### 壁纸日期与历史完整性

- 修复旧缓存会永久缺失 `2020-04-04` 清明壁纸的问题；内置快照现在会自动补齐缓存缺少的历史日期，2020 年 4 月可完整显示 30 张壁纸。
- Bing 官方 API 改用 `enddate` 作为中国区展示日期；缺失时将 `startdate` 后移一天，修复 7 月 14 日壁纸显示成 7 月 13 日的问题。
- 官方数据与历史归档出现同图跨日冲突时，以归档日期为准；同日期记录仍优先采用 Bing 官方的标题与版权信息。
- 新增归档完整性、历史缺日补齐、跨日同图和官方日期转换回归测试。

### 已下载壁纸缩略图

- 下载中心改为生成并缓存实际的小尺寸 JPEG 缩略图，不再让画廊直接长期解码和保留多张 4K 原图。
- 缩略图按源文件路径、大小与修改时间生成指纹；源图片变化后会自动失效并重新生成。
- 后台串行生成缩略图，限制图片尺寸与解码内存；损坏图片或损坏缓存会安全清理，不留下临时文件。
- 切换页面、刷新、删除壁纸和清空缓存时会同步取消任务、释放纹理并清理无主缩略图。

### 下载与更新安全

- 内嵌 aria2c.exe 每次释放为随机文件名，RPC 使用随机本机端口和随机 secret，退出后删除临时程序。
- 限制远程归档、aria2 RPC 与 Release API 响应体大小，避免异常响应占用过多内存。
- 应用内更新只接受 `bing-wallpaper-lib-vX.Y.Z-x64.exe` 形式的附件，并校验文件大小和 `MZ` 文件头。
- 更新完成后覆盖当前程序原路径再重启，避免开机自启注册表继续指向已删除的旧版本文件名。
